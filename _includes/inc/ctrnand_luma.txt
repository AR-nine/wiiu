Обратите внимание, что если у вас имеются другие файлы помимо `GodMode9.firm` в папке `/luma/payloads/` на SD-карте, удержание кнопки {% include inc/btn.md btn="START" %} при загрузке будет запускать "chainloader menu", где вам нужно будет использовать D-Pad и кнопку {% include inc/btn.md btn="A" %} для выбора "GodMode9" при выполнении этих инструкций.
{: .notice--info}

1. Запустите GodMode9, удерживая {% include inc/btn.md btn="START" %} во время включения приставки
1. Если вам предложат создать бэкап важных файлов, нажмите кнопку {% include inc/btn.md btn="A" %} чтобы сделать это, затем нажмите {% include inc/btn.md btn="A" %} чтобы продолжить после завершения
1. Нажмите {% include inc/btn.md btn="HOME" %}, чтобы попасть в меню действий
1. Выберите "Scripts..."
1. Выберите "setup_ctrnand_luma3ds"
1. Когда появится запрос, нажмите {% include inc/btn.md btn="A" %} для подтверждения
1. Нажмите {% include inc/btn.md btn="A" %}, чтобы разрешить запись в SysNAND (lvl1) и введите указанную комбинацию кнопок
1. Нажмите {% include inc/btn.md btn="A" %} чтобы продолжить
1. Нажмите {% include inc/btn.md btn="A" %} чтобы вновь заблокировать права на запись
