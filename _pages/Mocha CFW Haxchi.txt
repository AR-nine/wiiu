Mocha CFW - кастомная прошивка, которая патчит проверку подписей и загружает RedNAND.
{: .notice}

RedNAND - сокращение для "**Red**irected **NAND**" (перенаправление NAND). Это полная копия внутренней памяти консоли сохраненная на отдельном скрытом разделе SD карты. Эта копия использует вместо основной системной памяти. При использовании RedNAND основной физический модуль памяти консоли остается совершенно нетронутым. Это означает, что даже если вы окирпичите RedNAND копию прошивки, ваша консоль останется полностью работоспособна.
{: .notice--info}

Haxchi - это эксплойт, позволяющий запускать Mocha CFW прямо из меню home, без использования Homebrew Launcher.
{: .notice--info}

Если вам нужно купить одну из уязвимых игр, вопспользуйтесь кнопкой (Home), чтобы выйти из Homebrew Launcher. После покупки игры, заново войдите в [Homebrew Launcher](homebrew-launcher).
{: .notice--primary}

  **Ваша игра от DS virtual console, используемая для Haxchi, ДОЛЖНА быть ЛЕГАЛЬНОЙ копией!**

#### <a name="instructions" />Инструкция:

##### <a name="part1" />Часть I - Настройка Mocha CFW

1. Находясь в Homebrew Launcher, извлеките SD-карту из Wii U
2. Вставьте SD-карту приставки в компьютер
3. **Сделайте резервные копии всех файлов на SD карте, поскольку в следующем шаге карта будет отформатирована.**
4. Вставьте SD-карту обратно в Wii U
5. Запустите инсталлятор Haxchi
6. Выберите игру от DS virtual console, в которую вы собираетесь установить Haxchi, а затем нажмите (A) для подтверждения
7. Ознакомьтесь с появившимся предупреждением и нажмите (A) для установки
8. После окончания процесса, запустите вашу игру DS virtual console с установленным Haxchi
9. Должен появиться экран настройки Mocha CFW
10. Используя D-Pad ((ВВЕРХ) и (ВНИЗ) для перемещения по строкам и (ВЛЕВО) и (ВПРАВО) для изменения состояния пункта), выставьте следующие параметры:
  + Config view mode: **expert**
  + Skip this menu on launch: **on**
  + Show launch image: **on**
  + Don't relaunch OS: **off**
  + Launch home menu: **on**
  + redNAND: **on**  
  + SEEPROM redirection: **on**
  + OTP redirection: **on**
  + Use syshax.xml (coldboothax): **off**
11. Нажмите кнопку (A) для подтверждения выбора и выхода из меню
12. На экране Wii U должен появиться черный экран; это экран настройки RedNAND
13. Нажмите кнопку включения **на приставке** (не на контроллере) для того, чтобы начать копирование внутренней памяти консоли на SD-карту
  + Будьте готовы ждать; время копирование может занять несколько часов и зависит от количества свободного места во внутренней памяти, чем больше свободного места, тем быстрее будет идти копирование
14. По окончании процесса копирования, на экране Wii U появится черный экран
  + Это нормально; SD-карта была отформатирована
15. Вставьте SD-карту приставки в компьютер
  + Вы заметите, что размер SD-карты уменьшился, поскольку часть места отнал скрытый раздел с RedNAND
16. Скопируйте все файлы, которые вы скопировали с SD-карты на компьютер, обратно на SD-карту
17. Вставьте SD-карту обратно в Wii U
18. Включите Wii U

##### <a name="part2" />Часть II - Запуск Mocha CFW

1. На первой странице меню Home создайте папку и назовите ее "SysNAND"
2. Запустите Системные настройки (System Settings) и удалите все сетевые подключения
  + Необходимо сделать для того, чтобы избежать обновления SysNAND
  + **НЕОБХОДИМО сделать, поскольку в спящем режиме Wii U использует настройки интернет-соединения, указанные в SysNAND**
3. Запустите DS virtual console игру, в которой установлен Haxchi
4. Если эксплойт сработает успешно, запустится Mocha CFW и RedNAND
  + Если в меню Home вы не увидите папку SysNAND, значит RedNAND успешно загружен

___

В данный момент для загрузки RedNAND вам каждый раз будет нужно запускать Haxchi через игру для DS virtual console. На следующей странице мы установим CBHC (Coldboot Haxchi), который автоматизирует этот процесс.
{: .notice}

Если в меню home у вас нет папки SysNAND, можете быть уверены, что находитесь в RedNAND.
{: .notice--info}

    + Ничего не зажимать -> Загрузка в Mocha CFW в RedNAND
    + Зажать (A) -> Загрузка в Homebrew Launcher в SysNAND
    + Зажать (X) -> Загрузка в disc2app в SysNAND
