Mocha CFW - кастомная прошивка, которая патчит проверку подписей и загружает RedNAND.
{: .notice}

RedNAND - сокращение для "**Red**irected **NAND**" (перенаправление NAND). Это полная копия внутренней памяти консоли сохраненная на отдельном скрытом разделе SD карты. Эта копия использует вместо основной системной памяти. При использовании RedNAND основной физический модуль памяти консоли остается совершенно нетронутым. Это означает, что даже если вы окирпичите RedNAND копию прошивки, ваша консоль останется полностью работоспособна.
{: .notice--info}

Для загрузки патчей проверки подписи и RedNAND, каждый раз при запуске приставки, будет необходимо запускать Homebrew Launcher и из него Mocha CFW.
{: .notice--info}

#### <a name="instructions" />Инструкция:

##### <a name="part1" />Часть I - Настройка Mocha CFW

1. Находясь в Homebrew Launcher, извлеките SD-карту из Wii U
2. Вставьте SD-карту приставки в компьютер
3. **Сделайте резервные копии всех файлов на SD карте, поскольку в следующем шаге карта будет отформатирована.**
4. Вставьте SD-карту обратно в Wii U
5. Запустите Mocha CFW
6. Должен появиться экран настройки Mocha CFW
7. Используя D-Pad ((ВВЕРХ) и (ВНИЗ) для перемещения по строкам и (ВЛЕВО) и (ВПРАВО) для изменения состояния пункта), выставьте следующие параметры:
  + Config view mode: **expert**
  + Skip this menu on launch: **on**
  + Show launch image: **on**
  + Don't relaunch OS: **off**
  + Launch System Menu: **on**
  + redNAND: **on**  
  + SEEPROM redirection: **on**
  + OTP redirection: **on**
  + Use syshax.xml (coldboothax): **off**
8. Нажмите кнопку (A) для подтверждения выбора и выхода из меню
9. На экране Wii U должен появиться черный экран; это экран настройки RedNAND
10. Нажмите кнопку включения **на приставке** (не на контроллере) для того, чтобы начать копирование внутренней памяти консоли на SD-карту
  + Будьте готовы ждать; время копирование может занять несколько часов и зависит от количества свободного места во внутренней памяти, чем больше свободного места, тем быстрее будет идти копирование
11. По окончании процесса копирования, на экране Wii U появится черный экран
  + Это нормально; SD-карта была отформатирована
12. Вставьте SD-карту приставки в компьютер
  + Вы заметите, что размер SD-карты уменьшился, поскольку часть места отнал скрытый раздел с RedNAND
13. Скопируйте все файлы, которые вы скопировали с SD-карты на компьютер, обратно на SD-карту
14. Вставьте SD-карту обратно в Wii U
17. Включите Wii U

##### <a name="part2" />Часть II - Запуск Mocha CFW

1. На первой странице меню Home создайте папку и назовите ее "SysNAND"
2. Перейдите в Системные настройки (System Settings), Интернет (Internet), Подключение к Интернету (Connect to the Internet), нажмите (X) для того, чтобы отобразить имеющиеся подключения
3. Для каждого из имеющихся подключений (и для всех созданных в будущем), проделайте нижеследующие инструкции
  + Выберите "Изменить настройки" (Change Settings)
  + В пункте меню DNS выберите "Не получать автоматически" (Don't Auto-Obtain)
  + Введите указанные ниже адреса DNS
    + `168.235.092.108`
    + `081.004.127.020`
  + Нажмите "Подтвердить" (Confirm), а затем (B), чтобы сохранить внесенные изменения
  + Эти сервера заблокируют получение обновлений SysNAND на Wii U
4. Вернуться в меню home
5. Войдите в Homebrew Launcher, используя браузер
6. Запустите Mocha CFW
7. Если эксплойт сработает успешно, запустится Mocha CFW и RedNAND
  + Если в меню Home вы не увидите папку SysNAND, значит RedNAND успешно загружен

___

Вам придется каждый раз запускать Mocha CFW при запуске системы через Homebrew Launcher, чтобы войти в RedNAND
{: .notice}

Если в меню home у вас нет папки SysNAND, можете быть уверены, что находитесь в RedNAND.
{: .notice--info}

Если вы хотите пользоваться vWii, следует запускать его из SysNAND. При использовании RedNAND функциональность vWii недоступна полностью.
{: .notice--warning}
